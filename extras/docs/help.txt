--------------------------------------
Bash4LLM⁺ 2.x — Bash-first wrapper for the LLM
--------------------------------------
  --version            Bash4LLM version
  -h, --help           This Help

Usage:
  bash4llm "prompt"
  echo "text" | bash4llm
  bash4llm -f file.txt
  bash4llm --batch batch.txt

Options:

Input:
  -f <file>            Add file to FILE_INPUTS (repeatable)
  --json-input <json>  Direct JSON input (literal JSON, not a file)
  --template <name>    Template from bash4llm.d/templates
  --batch <file>       One prompt per non-empty line

Output:
  --text               Text output
  --raw                Raw output (no formatting)
  --json               Raw JSON output
  --pretty             Pretty-print JSON
  --no-color           Disable color output
  --out <path>         Save output to file or directory
  --save               Force save
  --nosave             Disable save
  --threshold <n>      Autosave threshold (bytes)

Streaming:
  --stream             Enable SSE streaming
  --no-stream          Disable streaming

Provider & Model:
  --provider <name>    Select provider
  --provider           Interactive provider selection
  -m, --model <name>   Select model
  --refresh-models     Refresh provider model list
  --list-models        List models (interactive)
  --list-models-raw    List models (raw, one per line)
  --list-providers     List providers (interactive)
  --list-providers-raw List providers (raw, one per line)
  --set-default <m>    Set persistent default model

Threads:
  --thread <id>        Enable persistent thread
  --thread-window [n]  Thread window size (default 10 if omitted)
  --init-thread        Safely initializes an empty thread (creating NDJSON and metadata)
                       and registers it in global thread index, without making API calls
                       Requires joint use with --thread <id>
                      
Generation parameters:
  --system <text>      System prompt
  --temperature <n>    Temperature (alias: --ture)
  --ture <n>           Temperature (internal alias)
  --max <n>            Max tokens

General:
  --dry-run            Show payload, skip API call
  --quiet              Reduce output
  --debug              Enable debug mode
  --chat               Interactive chat mode
  --bootstrap-only     Run bootstrap only

Extras installation:
  --install-extras           Install extras (all components)
  --install-extras <dir>     Install extras from directory
  --install-extras=<dir>     Inline form

Security & Vault:
  --vault              Manage encrypted API keys in the vault
                       (Interactive console tool)

  Unlock your vault session in RAM to bypass password prompts:
    `. ./bash4llm`                     (exports obfuscated _B4L_RT_CTX , wiped on exit)

  Bypass or disable the vault entirely:
    `BASH4LLM_VAULT_ENABLED=0`         (add to bash4llm.d/config to disable permanently)
    `export BASH4LLM_VAULT_ENABLED=0`     (disables vault for the active session)
    `BASH4LLM_VAULT_ENABLED=0 ./bash4llm` (disables vault for a single execution)

Examples:
  bash4llm "Summarize this text"
  echo "hello" | bash4llm --model mixtral-8x7b
  bash4llm -f prompt.txt --json
  bash4llm --stream "Generate streaming text"
  bash4llm --thread chat1 "Continue conversation"

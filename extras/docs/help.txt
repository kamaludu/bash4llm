--------------------------------------
GroqBash 2.0.0 — Bash-first wrapper for the Groq API
--------------------------------------
  --version            Groqbash version
  -h, --help           This Help

Usage:
  groqbash "prompt"
  echo "text" | groqbash
  groqbash -f file.txt
  groqbash --batch batch.txt

Options:

Input:
  -f <file>            Use file as input (repeatable)
  --json-input <file>  Direct JSON payload (use "-" for stdin)
  --template <name>    Template from groqbash.d/templates (requires {{CONTENT}})
  --batch <file>       One prompt per non-empty line

Output:
  --text               Text output (default)
  --raw                Raw output without newline
  --json               Raw JSON output
  --pretty             Pretty-print JSON
  --out <path>         Save output to file or directory
  --save / --nosave    Force save / disable save
  --threshold <n>      Autosave threshold (default 1000)

Streaming:
  --stream             Enable SSE streaming (provider must support it)
  --no-stream          Disable streaming

Provider & Model:
  --provider <name>    Select provider
  --provider           Interactive provider selection (TTY)
  -m, --model <name>   Select model
  --refresh-models     Refresh provider model list

Sessions:
  --session <id>       Enable persistent session
  --session-window <n> Number of messages to include (default 10)

Generation parameters:
  --system <text>      System prompt
  --temperature <n>    Temperature (alias: --ture)
  --max <n>            Max tokens

General:
  --dry-run            Show payload, skip API call
  --quiet              Reduce output
  --debug              Extended logging, preserve tmpdir

Immediate Actions:
  --list-models        List local models
  --list-providers     List available providers
  --set-default <m>    Set default model
  --install-extras     Install extras into runtime
  --show-config        Show active configuration
  --diagnostics        Run diagnostics

Examples:
  groqbash "Summarize this text"
  echo "hello" | groqbash --model mixtral-8x7b
  groqbash -f prompt.txt --json
  groqbash --stream "Generate streaming text"
  groqbash --session chat1 "Continue conversation"
